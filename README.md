# Hit & Blow Game 🎯

Java + Spring Bootで作成した数字当てゲームです。

## 🔗 アプリURL
https://hitandblow-1.onrender.com

※ 初回アクセス時は起動に少し時間がかかる場合があります

---

## 🎮 ゲーム概要

ランダムに生成された数字を推測し、
「Hit（数字と位置が一致）」
「Blow（数字のみ一致）」をヒントに正解を目指すゲームです。

---

## ✨ 機能

- 数字入力
- Hit & Blow判定
- 入力バリデーション（桁数・重複チェック）
- 履歴表示
- ギブアップ機能

---

## 🛠 使用技術

- Java
- Spring Boot
- Thymeleaf
- HTML / CSS / JavaScript
- SQLite
- MyBatis

---

## 💡 工夫した点

- 入力チェックを厳密に行い、ユーザーに分かりやすいエラーメッセージを表示
- UIをキーパッド形式にして操作性◎
- 画面遷移をシンプルにし、プレイしやすさを重視
